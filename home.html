1. Project description/overview
The Project objective is to develop a web search engine. It should be implemented using object-oriented analysis and design fundamentals. The search engine should allow users to search for information on the web. The Search Engine will accept a list of keywords and then return a list of URLs whose description contain or match the list of keywords.
The Search engine should be easy to understand, portable, user-friendly and much more.
